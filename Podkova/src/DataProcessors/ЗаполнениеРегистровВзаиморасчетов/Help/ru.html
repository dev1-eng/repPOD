Заполнение регистров взаиморасчетов

Обработка предназначена для ручного перезаполнения регистров взаиморасчетов, используемых при Онлайн распределении взаиморасчетов.

Для частичного перезаполнения:
Дата начала пересчета - дата с которой будут переформированы движения, если не указана то движения будут переформированы с начала ведения учета.
Ключ аналитики учета по партнерам и объект расчетов - что нужно перезаполнить.
