Сопоставление номенклатуры с рубрикатором

Форма имеет 2 основных режима работы: подготовка номенклатуры к выгрузке в Национальный каталог и сопоставление номенклатуры с рубрикатором 1С:Номенклатура. Режим подготовки к выгрузке включается в себя весь функционал сопоставления и дополнен инструментами для проверки и исправления проблем в заполнении данных номенклатуры (заполнение обязательных реквизитов).

Заполнение данных номенклатуры

Сопоставление с рубрикатором 1С:Номенклатура

Сопоставление необходимо для отображения информации в форме просмотра торгового предложения, возможности использования отборов при поиске торговых предложений, а так же для выгрузки данных в Национальный каталог.

Сопоставление значений реквизитов со значениями реквизитов рубрикатора 1С:Номенклатура

В зависимости от типов значений реквизитов информационной базы и значений реквизитов рубрикатора 1С:Номенклатуры сопоставление значений может происходить автоматически или вручную.

Порядок сопоставления:

  • Автоматическое сопоставление происходит для типов Число, Строка и значения Да/Нет. При этом в поле Сопоставлено указывается режим сопоставления Автоматически.
  • Ручное сопоставление происходит для ссылочных типов. При этом в колонке Тип значения указывается Список, в колонке Сопоставлено указывается информация о сопоставлении, например Сопоставлено 5 из 10. Для сопоставления значений необходимо нажать на гиперссылку и открыть окно Сопоставление значений реквизита рубрикатора 1С:Номенклатура, где производится сопоставление значений.

При ручном сопоставлении необходимо по гиперссылке открыть окно Сопоставление значений реквизита 1С:Номенклатура.
Выбор значения производится в выпадающем списке значений в колонке Значение 1С:Номенклатура. По команде Заполнить сервис подставляет совпадающие по наименованию значения.

Сопоставление реквизитов вида номенклатуры с реквизитами рубрикатора 1С:Номенклатура

Список реквизитов содержит доступные для сопоставления реквизиты номенклатуры, дополнительные реквизиты номенклатуры, дополнительные реквизиты характеристик номенклатуры, общие реквизиты номенклатуры, общие реквизиты характеристик.

Порядок сопоставления:

  • В списке Видов номенклатуры выбрать необходимую строку, в которой произведено сопоставление с категорией рубрикатора 1С:Номенклатура.
  • В правой части формы заполнится список доступных реквизитов рубрикатора 1С:Номенклатура и сопоставленных ранее реквизитов. Для сопоставленных значений реквизитов указывается информация о сопоставлении, например, Сопоставлено 5 из 10.
  • В нужной строке списка реквизитов в колонке Реквизит вида номенклатуры выбрать реквизит из выпадающего списка.

Сопоставление элементов номенклатуры с категориями рубрикатора 1С:Номенклатура

В режиме сопоставления по элементам номенклатуры категории рубрикатора 1С:Номенклатура сопоставляются с элементами справочника номенклатура. Возможно сопоставление общих реквизитов номенклатуры и по дополнительным реквизитам.

Сопоставление по элементам номенклатуры имеет приоритет над сопоставленной группой или видом номенклатуры.

Если сопоставление по номенклатуре не выполнено, используется сопоставление по группе номенклатуры, по виду номенклатуры.

Сопоставление групп номенклатуры с категориями рубрикатора 1С:Номенклатура

В режиме сопоставления по группам номенклатуры категории рубрикатора 1С:Номенклатура сопоставляются с группами номенклатуры по иерархии. Возможно сопоставление только общих реквизитов номенклатуры.

Сопоставление по группе номенклатуры имеет приоритет перед сопоставленным видом номенклатуры.

Сопоставление видов номенклатуры с категориями рубрикатора 1С:Номенклатура

Сопоставление видов номенклатуры необходимо для поиска товаров по категориям рубрикатора 1С:Номенклатура.

Выгрузка торговых предложений производится по сопоставленным видам номенклатуры производится в рубрики сервиса 1С:Номенклатура. Остальные торговые предложения публикуются в служебной рубрике Разное.
Для выгрузки данных номенклатуры в Национальный каталог сопоставление с рубрикатором является обязательным, несопоставленная номенклатура выгружена не будет.

Порядок сопоставления:

  • В строке с необходимым Видом номенклатуры двойным кликом в колонке Категория 1С:Номенклатура открыть окно Выбор категории рубрикатора 1С:Номенклатура.
  • Найти в дереве нужную категорию.
  • Нажать команду Выбрать.

Заполнение данных номенклатуры

Таблица Заполнение данных содержит перечень проблем заполнения. В колонке Описание в виде гиперссылки выводится текстовое описание проблемы. Клик по гиперссылке открывает форму элемента Номенклатуры и выводит поясняющие сообщения. После устранения проблем заполнения в ячейке Описание появляется текст Исправлено.
