Настройка загрузки номенклатуры множественная

Форма предназначена для выбора вида номенклатуры при загрузке нескольких позиций номенклатуры, категории по которым не были загружены/сопоставлены. Если для номенклатуры ведутся характеристик форма позволяет выбрать режим загрузки характеристик, а также выбрать характеристики для загрузки.

Если характеристики по номенклатуре не ведутся, форма открыватся в режиме одной страницы, на которой необходимо выбрать, сопоставить или создать виды номенклатуры для каждой строки номенклатуры.

Для автоматического создания видов номенклатуры для всех строк номенклатуры необходимо нажать кнопку Создать виды номенклатуры автоматически.

Если по номенклатуре ведутся характеристики, форма открывается в режиме двух страниц. На первой странице необходимо выбрать, сопоставить или создать виды номенклатуры для каждой строки номенклатуры. На второй странице необходимо выбрать режимы загрузки характеристик, а также выбрать характеристики для загрузки.
