Распределение расходов на себестоимость продукции

Документ предназначен для распределения постатейных расходов на себестоимость продукции. Для выполнения распределения в списке отражены нераспределенные расходы с детализацией по статьям расходов и аналитике расходов.

Для настройки распределения двойным щелчком необходимо вызвать окно настроек, либо выделить перечень строк в списке расходов и нажать кнопку "Сформировать документы распределения". Документ по умолчанию заполняется на основании правила распределения расходов заданного в статье расходов.

Постатейный расход может быть распределен:

  • на партии - постатейный расход распределяется на партии производства. Возможно распределение между партиями по правилу, либо на перечисленные вручную;
  • на другие статьи расходов - возможность отнести постатейный расход на статьи расходов с другим вариантом распределения, например в целях переквалификации расхода;
  • на НЗП - если постатейный расход не требуется распределять в месяце распределения, то возможно оставить расход в незавершенном производстве.

В зависимости от того куда распределяется постатейный расход в документе меняется состав закладок документа, позволяя указывать детальные данные для распределения. Для каждого направления распределения требуется указать долю стоимость, за исключением если выбрано только одно направление распределения.

Правила распределения

На закладке можно настроить правило распределения, по которому будут распределяться расходы между подразделениями и партиями производства.
Распределение выполняется в 2 шага:

  • постатейный расход распределяется между подразделениями;
  • на партии производства подразделений.

Подразделения, на которые будет распределен постатейный расход, можно перечислить вручную с указанием долей стоимости на каждое подразделение, либо выбрать из списка:

  • все подразделениям;
  • текущее - подразделение документа, указанное на закладке Основное;
  • нижестоящие - нижестоящие подразделения в справочнике Структура предприятия, подразделения документа;
  • вышестоящее - вышестоящее подразделения в справочнике Структура предприятия, подразделения документа;
  • указанные - позволяет указать перечень подразделений вручную, без указания долей стоимости.

Распределение между партиями производства и подразделениями может быть выполнено по общему правилу. В этом случае подразделения определяются правилом распределения между партиями с соответствующим отбором по подразделениям. Для применения такого варианте требуется выбрать вариант Распределять по единой базе для партий и подразделений. Если же требуется сумму расхода распределить по подразделениям по одному правилу, а внутри подразделений применить другое правило, то следует выбрать настройку Распределять по разным базам для партий и подразделений.

Доступные базы распределения между подразделениями:

  • задается вручную - на форме появляется закладка Подразделения;
  • по показателю - указывается в дополнительном поле Показатель;
  • стоимость материалов;
  • сумма оплаты труда;
  • стоимость материалов и сумма оплаты труда.

Доступные базы распределения по партиям:

  • стоимость материалов и сумма оплаты труда;
  • стоимость материалов;
  • сумма оплаты труда;
  • нормативная оплата труда;
  • количество продукции с/без учета будущих выпусков;
  • количество видов работ;
  • количество материалов;
  • вес продукции с/без учета будущих выпусков;
  • вес материалов;
  • объем продукции с/без учета будущих выпусков;
  • объем материалов;
  • плановая стоимость продукции с/без учета будущих выпусков.

Для баз распределения по продукции возможно распределение с учетом будущих выпусков или без, т.е. постатейный расход будет включаться по мере выпуска продукции или будет полностью отнесен на выпуск месяца распределения.
Для остальных баз распределения постатейный расход всегда включается в стоимость продукции по мере выпуска продукции.

Этапы заказов

Содержит таблицу, в которой можно указать этапы заказов, на которые можно отнести расходы в указанной доле стоимости.

Выпуски без распоряжений

Содержит таблицу, в которой указываются документ выпуска без распоряжений и код строки, на которые необходимо отнести распределяемые расходы в указанной доле стоимости

Партии производства

Содержит информацию о партиях производства, на которые нужно отнести указанную долю стоимости распределяемой суммы расходов.
Добавление новых строк выполняется посредством формы подбора партий производства.

В качестве партий производства выступают:

  • Этапы производства
  • Документы производства без заказа
  • Отчеты переработчиков

При этом, для этапов производства доступны для выбора промежуточные этапы, входящие в одну партию производства (партией является выпускающий этап).

Другие статьи расходов

Возможность отнесения затрат накопленных по распределяемой статье на статьи с другим вариантом распределения. Данная возможность может быть использована для переквалификации расходов.

Указание доли стоимости необходимо для предварительного распределения расходов до момента расчета фактической себестоимости расхода. Фактическая себестоимость расхода рассчитывается после выполнения всех операций по закрытию месяца. После расчета фактической себестоимости данные по распределению расходов будут уточнены.
